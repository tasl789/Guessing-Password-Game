Password Guessing Game

This project is a Password Guessing Game developed using Python. The game randomly selects a secret word, and the user must guess it correctly.

Features

  • Three difficulty levels: Easy, Medium, and Hard
  • Randomly generated passwords based on the selected level
  • Hint system to guide the user after each incorrect guess
  • Attempts counter to track the number of guesses
  • Interactive command-line gameplay

How the Game Works

  • The user selects a difficulty level
  • A secret password is randomly chosen
  • The user enters guesses until the correct password is found
  • The game displays the total number of attempts

Purpose

This project was created to practice Python programming concepts such as loops, conditional statements, string handling, and the random module.

Technologies Used
